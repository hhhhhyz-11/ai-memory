### 后续步骤 (Step 2-15)

每步骤重复以下流程（替换版本号）:

```bash
# 变量：当前版本 CURRENT=11.11.8, 目标版本 NEXT=12.0.12
CURRENT=11.11.8
NEXT=12.0.12
NAME=gitlab_${NEXT}

# 1. 备份
docker exec -it gitlab_$CURRENT gitlab-rake gitlab:backup:create

# 2. 停止并删除
docker stop gitlab_$CURRENT
docker rm gitlab_$CURRENT

# 3. 拉取新镜像
docker pull gitlab/gitlab-ce:${NEXT}-ce.0

# 4. 启动
docker run --detach \
 --publish 8080:80 --publish 4443:443 --publish 2222:22 \
 --name $NAME \
 --restart always \
 --volume /home/gitlab/data/data:/var/opt/gitlab \
 --volume /home/gitlab/log:/var/log/gitlab \
 --volume /home/gitlab/etc:/etc/gitlab \
 gitlab/gitlab-ce:${NEXT}-ce.0

# 5. 等待启动后检查
docker exec -it $NAME gitlab-rake gitlab:check
docker exec -it $NAME gitlab-rake cache:clear
```
