5. **SonarQube Issues 批量导出**
   - 背景：SonarQube 页面导出 PDF 超过 10000 条限制
   - 方案：导出 Protobuf 格式，用 Java 工具转 CSV
   - 解决：
     - 用户提供了 SonarPbToCsvConverter Java 代码
     - 克隆 GitLab 项目 `test-tool`，用 Maven 打包
     - 下载 commons-csv 依赖
     - 手动测试成功（19874 issues）
   - 流程：
     1. SonarQube 页面手动导出项目（Protobuf 格式）→ 生成 .zip
     2. Jenkins Pipeline 调用 shell 脚本
     3. 脚本检测 zip → 解压 → 删除源文件 → 转 CSV → 输出到 csv 目录
     4. Nginx 搭建文件服务器，提供下载入口
   - 最终效果：Jenkins 选择项目 → 执行 → 输出下载链接
