# 2026-03-20 工作日志

## GitLab CI + SonarQube 配置优化

1. **自动合并 MR 问题**
   - 问题：`when: manual` 导致流水线不会自动合并
   - 解决：去掉手动触发，改为自动执行

2. **合并返回 405 错误**
   - 原因：MR 状态不允许合并或 squash 配置问题
   - 解决：合并 API 加参数 `"squash": false`

3. **interruptible 参数不支持**
   - 问题：GitLab 11.1.4 不支持 `interruptible` 键
   - 解决：去掉该参数

4. **重复流水线问题**
   - 问题：提交代码时，旧 pipeline 还在运行，新 pipeline 等待
   - 解决：在 job 脚本中用 GitLab API 取消同一分支的旧 pipeline
   - API：`POST /projects/:id/pipelines/:pipeline_id/cancel`
