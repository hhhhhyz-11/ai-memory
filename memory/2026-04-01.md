2. **GitLab CI + SonarQube 覆盖率问题排查**
   - 用户反馈：CI 流水线 SonarQube 覆盖率始终为 0
   - 分析其 .gitlab-ci.yml 配置，发现 Maven 命令缺少 JaCoCo 相关参数
   - 提供详细修复方案：
     - 添加 `-Dsonar.java.binaries=target/classes`
     - 添加 `-Dsonar.coverage.jacoco.xmlReportPaths=target/site/jacoco/jacoco.xml`
     - 不能使用 `-Dmaven.test.skip=true`（会跳过测试导致无覆盖率）
     - pom.xml 需要配置 jacoco-maven-plugin
   - 提供了排查步骤：执行 `mvn clean test jacoco:report` 检查 target/jacoco.exec 是否生成
