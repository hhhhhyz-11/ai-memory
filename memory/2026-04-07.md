- 部署类型：单机部署

---

### 步骤 1：创建用户 ✅
```bash
groupadd kingbase
useradd -g kingbase -m -d /home/kingbase -s /bin/bash kingbase
echo "kingbase:kingbase" | chpasswd
id kingbase
# uid=1000(kingbase) gid=1000(kingbase) groups=1000(kingbase)
```
**备注**：home 目录已存在，skel 目录未复制文件（不影响）
