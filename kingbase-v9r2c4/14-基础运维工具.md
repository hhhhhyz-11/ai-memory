# KingbaseES V9R2C14 基础运维工具

> 本文档介绍 KingbaseES V9R2C14 自带的基础运维工具及其使用方法。

## 一、数据库启停工具

### 1.1 sys_ctl

KingbaseES 数据库服务控制工具。

```bash
# 启动数据库
sys_ctl -D $KINGBASE_DATA start

# 停止数据库（快速关闭）
sys_ctl -D $KINGBASE_DATA stop -m fast

# 优雅关闭（等待连接断开）
sys_ctl -D $KINGBASE_DATA stop -m smart

# 立即停止（不推荐，可能导致数据丢失）
sys_ctl -D $KINGBASE_DATA stop -m immediate

# 重启数据库
sys_ctl -D $KINGBASE_DATA restart

# 重新加载配置文件
sys_ctl -D $KINGBASE_DATA reload

# 查看数据库状态
sys_ctl -D $KINGBASE_DATA status

# 查看所有配置参数
sys_ctl -D $KINGBASE_DATA show all

# 查看单个参数
sys_ctl -D $KINGBASE_DATA show max_connections
sys_ctl -D $KINGBASE_DATA show shared_buffers
```

### 1.2 kingbase 命令

```bash
# 启动数据库
kingbase start -D $KINGBASE_DATA

# 停止数据库
kingbase stop -D $KINGBASE_DATA

# 重启数据库
kingbase restart -D $KINGBASE_DATA

# 查看版本
kingbase --version

# 查看帮助
kingbase --help
```

## 二、连接工具 ksql

### 2.1 基本用法

```bash
# 本地连接
ksql -U system -d test

# 指定主机和端口
ksql -U system -d test -h 192.168.1.100 -p 54321

# 连接时执行命令
ksql -U system -d test -c "SELECT version();"
ksql -U system -d test -f /path/to/script.sql

# 输出格式
ksql -U system -d test -X              # 不读取 .sqlrc
ksql -U system -d test -H              # HTML 输出
ksql -U system -d test -t              # 不输出列名
ksql -U system -d test -A              # 非对齐输出模式
ksql -U system -d test -F '|'          # 指定分隔符
```

### 2.2 ksql 交互命令

```sql
-- 在 ksql 交互界面中使用

\list
\l
\l+

-- 连接到其他数据库
```

## 三、备份恢复工具

### 3.1 sys_dump（逻辑备份）

```bash
# 备份整个数据库
sys_dump -U system -d test -f /tmp/test_backup.sql

# 备份为压缩格式
sys_dump -U system -d test -Fc -f /tmp/test_backup.dump

# 备份指定表
sys_dump -U system -d test -t table_name -t another_table -f /tmp/tables_backup.sql

# 只备份表结构（不包括数据）
sys_dump -U system -d test --schema-only -f /tmp/schema.sql

# 只备份数据（不包括结构）
sys_dump -U system -d test --data-only -f /tmp/data.sql

# 备份所有数据库
sys_dumpall -U system -f /tmp/all_databases.sql

# 备份所有角色
sys_dumpall -U system --roles-only -f /tmp/roles.sql
```

### 3.2 sys_restore（恢复）

```bash
# 恢复纯 SQL 备份
ksql -U system -d test -f /tmp/test_backup.sql

# 恢复压缩备份
sys_restore -U system -d test /tmp/test_backup.dump

# 创建新数据库并恢复
sys_restore -U system -C -d postgres /tmp/test_backup.dump

# 列出备份内容
sys_restore -l /tmp/test_backup.dump

# 只恢复指定表
sys_restore -U system -d test -t table_name /tmp/test_backup.dump
```

### 3.3 备份脚本示例

```bash
#!/bin/bash
# backup.sh - 数据库备份脚本

BACKUP_DIR="/data/backup/kingbase"
DATE=$(date +%Y%m%d_%H%M%S)
DB_HOST="192.168.1.100"
DB_PORT="54321"
DB_NAME="test"
DB_USER="system"
RETENTION_DAYS=30

mkdir -p $BACKUP_DIR

# 执行备份
sys_dump -U $DB_USER -d $DB_NAME -h $DB_HOST -p $DB_PORT \
    -Fc -f $BACKUP_DIR/${DB_NAME}_${DATE}.dump

# 备份角色
sys_dumpall -U $DB_USER -h $DB_HOST -p $DB_PORT \
    --roles-only -f $BACKUP_DIR/roles_${DATE}.sql

# 压缩备份
gzip $BACKUP_DIR/${DB_NAME}_${DATE}.dump

# 清理过期备份
find $BACKUP_DIR -name "*.dump.gz" -mtime +$RETENTION_DAYS -delete
find $BACKUP_DIR -name "roles_*.sql" -mtime +$RETENTION_DAYS -delete

echo "备份完成: $BACKUP_DIR/${DB_NAME}_${DATE}.dump.gz"
```

## 四、系统视图查询工具

### 4.1 连接相关

```sql
-- 查看当前连接
SELECT * FROM sys_stat_activity;

-- 查看连接数统计
SELECT 
    datname,
    numbackends,
    xact_commit,
    xact_rollback,
    blks_hit,
    blks_read
FROM sys_stat_database;

-- 查看长时间运行的查询
SELECT 
    pid,
    now() - query_start AS duration,
    state,
    query
FROM sys_stat_activity
WHERE state != 'idle'
ORDER BY query_start;

-- 终止长时间运行的查询
SELECT sys_terminate_backend(pid);
```

### 4.2 性能相关

```sql
-- 查看缓存命中率
SELECT 
    datname,
    blks_hit,
    blks_read,
    ROUND(blks_hit * 100.0 / (blks_hit + blks_read), 2) AS hit_ratio
FROM sys_stat_database;

-- 查看 Top SQL
SELECT 
    queryid,
    calls,
    total_exec_time,
    mean_exec_time,
    rows
FROM sys_stat_statements
ORDER BY total_exec_time DESC
LIMIT 10;

-- 查看等待事件
SELECT 
    wait_event_type,
    wait_event,
    COUNT(*)
FROM sys_stat_activity
WHERE wait_event IS NOT NULL
GROUP BY wait_event_type, wait_event;
```

### 4.3 表和索引相关

```sql
-- 查看表大小
SELECT 
    schemaname,
    relname,
    pg_size_pretty(pg_total_relation_size(schemaname||'.'||relname)) AS total_size,
    pg_size_pretty(pg_relation_size(schemaname||'.'||relname)) AS table_size
FROM sys_stat_user_tables
ORDER BY pg_total_relation_size(schemaname||'.'||relname) DESC;

-- 查看索引使用情况
SELECT 
    schemaname,
    relname,
    indexrelname,
    idx_scan,
    idx_tup_read,
    idx_tup_fetch
FROM sys_stat_user_indexes
ORDER BY idx_scan DESC;

-- 查看未使用的索引
SELECT 
    schemaname,
    relname,
    indexrelname
FROM sys_stat_user_indexes
WHERE idx_scan = 0
AND indexrelname NOT LIKE '%pkey';
```

### 5.3 实时监控脚本

```bash
#!/bin/bash
# monitor.sh - 实时监控脚本

DB_NAME="test"
DB_USER="system"

echo "=== KingbaseES 实时监控 ==="
echo "时间: $(date)"
echo ""

# 连接数
echo "--- 连接统计 ---"
ksql -U $DB_USER -d $DB_NAME -c "
SELECT 
    COUNT(*) AS current_connections,
    COUNT(*) FILTER (WHERE state = 'active') AS active_connections,
    COUNT(*) FILTER (WHERE state = 'idle') AS idle_connections
FROM sys_stat_activity;
"

# 缓存命中率
echo ""
echo "--- 缓存命中率 ---"
ksql -U $DB_USER -d $DB_NAME -c "
SELECT 
    ROUND(blks_hit * 100.0 / (blks_hit + blks_read), 2) AS cache_hit_ratio,
    blks_hit,
    blks_read
FROM sys_stat_database 
WHERE datname = '$DB_NAME';
"

# TPS
echo ""
echo "--- 事务统计 ---"
ksql -U $DB_USER -d $DB_NAME -c "
SELECT 
    xact_commit AS commits,
    xact_rollback AS rollbacks,
    xact_commit + xact_rollback AS total_transactions
FROM sys_stat_database 
WHERE datname = '$DB_NAME';
"

# 长时间查询
echo ""
echo "--- 长时间运行的查询 ---"
ksql -U $DB_USER -d $DB_NAME -c "
SELECT 
    pid,
    now() - query_start AS duration,
    state,
    SUBSTRING(query, 1, 80) AS query_preview
FROM sys_stat_activity
WHERE state != 'idle'
AND now() - query_start > interval '1 minute'
ORDER BY query_start;
"
```

## 七、日志管理

### 7.1 日志相关配置

```bash
vim $KINGBASE_DATA/kingbase.conf

# 开启日志收集
logging_collector = on
log_directory = 'log'
log_filename = 'kingbase-%Y-%m-%d.log'

# 日志级别
log_min_messages = warning
log_min_error_statement = error

# 记录内容
log_statement = 'none'           # 不记录 SQL
log_statement = 'ddl'            # 只记录 DDL
log_statement = 'mod'            # DDL + DML
log_statement = 'all'            # 记录所有 SQL

# 慢查询日志
log_min_duration_statement = 1000  # 记录超过 1 秒的查询

# 重载配置
sys_ctl -D $KINGBASE_DATA reload
```

### 7.2 日志分析

```bash
# 查看最近错误
grep -i error $KINGBASE_DATA/log/*.log | tail -50

# 查看慢查询
grep "duration" $KINGBASE_DATA/log/*.log | tail -20

# 统计错误类型
grep -i error $KINGBASE_DATA/log/*.log | awk -F: '{print $3}' | sort | uniq -c

# 查看连接日志
grep "connection received" $KINGBASE_DATA/log/*.log | tail -20
```

## 八、锁管理

### 8.1 查看锁

```sql
-- 查看当前锁
SELECT 
    locktype,
    database,
    relation::regclass,
    page,
    tuple,
    virtualxid,
    transactionid,
    classid,
    objid,
    objsubid,
    virtualtransaction,
    pid,
    mode,
    granted,
    fastpath
FROM sys_locks;

-- 查看阻塞会话
SELECT 
    blocked_locks.pid AS blocked_pid,
    blocked_activity.usename AS blocked_user,
    blocking_locks.pid AS blocking_pid,
    blocking_activity.usename AS blocking_user,
    blocked_activity.query AS blocked_statement,
    blocking_activity.query AS blocking_statement,
    blocked_activity.state AS blocked_state
FROM sys_stat_activity blocked_activity
JOIN sys_locks blocked_locks ON blocked_activity.pid = blocked_locks.pid
JOIN sys_stat_activity blocking_activity ON blocked_locks.granted 
    AND blocking_activity.pid = blocked_locks.pid
WHERE blocked_activity.state != 'idle';
```

### 8.2 处理锁

```sql
-- 终止持有锁的会话
SELECT sys_terminate_backend(pid);

-- 终止被锁阻塞的会话
SELECT sys_cancel_backend(pid);
```

## 九、常用运维脚本

### 9.1 健康检查脚本

```bash
#!/bin/bash
# health_check.sh - 数据库健康检查

DB_USER="system"
DB_NAME="test"

echo "=== KingbaseES 健康检查 ==="
echo "时间: $(date)"
echo ""

# 1. 数据库连接测试
echo "1. 数据库连接测试"
ksql -U $DB_USER -d $DB_NAME -c "SELECT 1;" > /dev/null 2>&1
if [ $? -eq 0 ]; then
    echo "   ✓ 数据库连接正常"
else
    echo "   ✗ 数据库连接失败"
fi

# 2. 表空间检查
echo ""
echo "2. 表空间使用率"
ksql -U $DB_USER -d $DB_NAME -c "
SELECT 
    spcname AS tablespace,
    pg_size_pretty(pg_tablespace_size(spcname)) AS size
FROM pg_tablespace;
"

# 3. 复制状态
echo ""
echo "3. 复制状态"
REPL_COUNT=$(ksql -U $DB_USER -d $DB_NAME -t -c "SELECT COUNT(*) FROM sys_stat_replication;")
if [ "$REPL_COUNT" -gt 0 ]; then
    echo "   ✓ 复制正常 (连接数: $REPL_COUNT)"
else
    echo "   ⚠  无复制连接（单实例模式？）"
fi

# 4. 缓存命中率
echo ""
echo "4. 缓存命中率"
ksql -U $DB_USER -d $DB_NAME -c "
SELECT 
    ROUND(blks_hit * 100.0 / (blks_hit + blks_read), 2) AS hit_ratio
FROM sys_stat_database WHERE datname = '$DB_NAME';
"

echo ""
echo "=== 检查完成 ==="
```

### 9.2 批量执行脚本

```bash
#!/bin/bash
# batch_sql.sh - 批量执行 SQL

DB_USER="system"
DB_NAME="test"
SQL_DIR="/opt/sql_scripts"

# 执行目录下的所有 SQL 文件
for sql_file in $SQL_DIR/*.sql; do
    filename=$(basename "$sql_file")
    echo "执行: $filename"
    
    ksql -U $DB_USER -d $DB_NAME -f "$sql_file" > /tmp/${filename}.log 2>&1
    
    if [ $? -eq 0 ]; then
        echo "  ✓ 成功"
    else
        echo "  ✗ 失败，请查看: /tmp/${filename}.log"
    fi
done
```
