## 二、核心理论范畴（6 大支柱）

### 1. 系统辨识与信息处理
- 从噪声中提取本质信息：滤波、预测、相关处理、模式识别
- 建模理论：数学模型（代数/微分/积分方程、概率论、模糊数学）
- **核心启示**：对 AI 而言 = **状态估计 + 环境感知**

### 2. 最优控制
- 以最小代价达到控制目的
- 工具：线性规划、动态规划、极小值原理、最优化技术
- **核心启示**：对 AI 而言 = **决策优化（资源/路径/策略）**

### 3. 自我进化（自适应/自学习）
- 自稳定系统 → 自适应控制 → 自学习系统 → 专家系统 → 自组织系统 → 智能系统
- **核心启示**：对 AI 而言 = **自我改进 + 知识积累 + 能力扩展**

### 4. 容错系统
- 冗余技术（备份）
- 自诊断理论：定期检查、故障定位、自动切换备份
- 纠错编码：自动发现并纠正错误
- **核心启示**：对 AI 而言 = **错误处理 + 降级策略 + 自我修复**

### 5. 仿真技术
- 物理仿真 → 模拟计算机 → 数字计算机 → 混合计算机
- **核心启示**：对 AI 而言 = **沙箱测试 + 模拟推演 + 预执行验证**

### 6. 应用领域扩展
- 技术领域：机械、电力、航空航天
- 社会延伸：社会控制论、经济控制论、生物控制论、军事控制论、人口控制论
- **核心启示**：控制论思想可跨领域泛化

## 三、关键工程控制论概念映射到 AI Agent

| 控制论概念 | AI Agent 对应 |
|-----------|--------------|
| 反馈（Negative Feedback） | 观察环境状态 → 调整行动（ReAct/Reflexion） |
| 传递函数 | 状态转移模型 / 世界模型 |
| 伺服系统 | Goal-seeking Agent（目标驱动） |
| 解耦控制 | 多任务分离，独立子目标处理 |
| 随机输入下的线性系统 | 不确定环境下的决策（Bayesian/POMDP） |
| 继电器伺服（bang-bang） | 离散动作空间的快速搜索 |
| 非线性系统 | 复杂涌现行为 |
| 摄动理论 | 对扰动的鲁棒性分析 |
| 最优控制 | RL / 策略优化 |
| 噪声过滤 | 不确定性量化 / 安全过滤 |
| 自行镇定 | 自我修复 / 自我调试 |
| 适应环境 | Online Learning / Continual Learning |
| 误差控制 | 自我评估 + 错误纠正（Self-Correction） |

## 四、钱学森方法论要点

1. **从定性到定量**：把工程经验提炼为科学理论
2. **一般性与特殊性**：抽取共性，提高为科学理论，再用以指导千差万别的实践
3. **设计与实验结合**：理论→设计→仿真→实验→迭代
4. **系统观**：整体大于部分之和，强调总体设计
5. **闭环思维**：信息反馈是控制的核心
