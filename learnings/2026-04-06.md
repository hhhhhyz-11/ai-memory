## 📚 技术知识

### 钉钉自定义机器人 Webhook
- 通过 Webhook URL 发送消息，支持 Markdown 格式
- 关键词验证：必须包含设定的关键词才能发送成功
- 适用场景：日报/周报/月报自动推送

### OpenClaw Agent 批量清理
- 可通过脚本批量删除冗余 Agent
- 清理后配置文件从 303KB 降至 85KB
- 注意：写回配置时要保留历史字段，避免丢失其他配置

### Gateway 服务安装
- 需执行 `openclaw gateway install` 安装服务
- 再执行 `openclaw gateway start` 启动
- 建议检查服务状态确保开机自启

## ⚠️ 踩坑记录

1. **配置文件清理后字段丢失**
   - 问题：批量清理 Agent 后，部分历史配置字段丢失
   - 原因：清理时整个 agents 配置写回，丢失了其他配置节
   - 教训：修改配置前先完整备份，清理时只删除 agents 节不要影响其他配置

2. **qqbot 全局插件与内置插件冲突**
   - 问题：global plugin 被 bundled plugin 覆盖
   - 影响：不影响功能但有警告信息
   - 状态：待进一步排查

## 💡 经验教训

- **配置文件修改前务必完整备份**，包括所有配置节
- 批量清理操作建议分步骤执行，每次只改一个配置节
- Gateway 服务安装后需验证是否开机自启
- 钉钉 Webhook 消息必须包含关键词才能发送成功
