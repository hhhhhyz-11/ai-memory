# 2026-03-11 技术总结

## 定时任务修复

- **问题**：定时任务 payload 写死日期"今天是2026-03-09"
- **原因**：0点执行应读取"昨天"而非固定日期
- **解决**：改为动态描述"读取昨天的日志"

## GitLab CI + SonarQube 流水线

- **问题**：CI 分成 build + sonar 两个 stage，sonar 阶段无 .class 文件
- **问题**：sonar-scanner 扫描 Java 需要 `sonar.java.binaries` 指定编译产物
- **解决**：合并为单一 job，Maven 编译和 Sonar 分析一起执行
- **优化参数**：
  - `-T 2C` 并行编译
  - `-Dmaven.test.skip=true` 跳过测试
  - `-Dsonar.exclusions` 排除目录

## SonarQube Issues 批量导出

- **场景**：页面导出 PDF 超过 10000 条限制
- **方案**：导出 Protobuf → 用 Java 工具转 CSV
- **工具**：SonarPbToCsvConverter + commons-csv
- **最终流程**：
  1. SonarQube 手动导出项目（Protobuf 格式）→ .zip
  2. Jenkins Pipeline 调用 shell 脚本
  3. 脚本：解压 → 删除源文件 → 转 CSV → 输出到 csv 目录
  4. Nginx 搭建文件服务器提供下载

- **验证**：手动测试成功转换 19874 issues
