## 📚 技术知识

### 1. Memory Core Dreaming 机制问题

**问题背景：**
- Dreaming 功能超时后不停重试，每次创建 3 个 session（light/REM/deep）
- 连续失败后 session 堆积成灾
- sessions.json 无 maxEntries 限制、无过期删除机制

**关键配置修复：**
```json
"session": {
  "maintenance": {
    "mode": "enforce",
    "maxEntries": 200,
    "pruneAfter": "30d",
    "rotateBytes": "50mb"
  }
}
```

### 2. Cron 定时任务创建

- Job ID 用于唯一标识任务
- 任务可手动触发验证
- 执行时间统一使用北京时间（任务调度器已处理时区转换）
